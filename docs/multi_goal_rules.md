# Multi Goal Rules (MVP version)

В MVP версии бота поддерживается только одна активная цель на пользователя.

## Основные правила

1. У пользователя может быть только одна цель со статусом `active`.

2. Пользователь НЕ может создать новую цель, если у него уже есть активная цель.

3. Чтобы создать новую цель, пользователь должен:
   - завершить текущую цель (`completed`)
   - или отменить цель (`cancelled`)

4. История целей сохраняется:
   - пользователь может иметь несколько целей в базе
   - но работать может только с одной активной

5. Переключение между целями отсутствует:
   - нет `active_goal_id` переключения
   - нет multi-goal контекста

6. Все действия пользователя всегда относятся к одной текущей активной цели.

## Упрощение логики

В MVP:
- нет необходимости в сложном chat context
- нет необходимости в goal switching
- AI всегда работает с одной целью

## Возможное расширение в будущем

В будущем можно добавить:
- несколько активных целей
- переключение между целями
- отдельные сессии для каждой цели
