# AI Goal Tracker Telegram Bot

Telegram-бот на основе AI, который помогает пользователю достигать целей через персональный план, ежедневные отчеты и фотопруфы выполнения.

## Основная идея

Пользователь:
1. запускает бота
2. задает цель
3. отвечает на уточняющие вопросы
4. получает персональный план
5. выполняет шаги плана
6. отправляет фотопруфы или скриншоты выполнения
7. получает ежедневный прогресс и недельные summary

## Основные возможности

- создание одной цели
- отдельный контекст для каждой цели
- AI-генерация плана достижения цели
- корректировка плана по запросу пользователя
- ежедневные check-ins
- proof validation по фото и скриншотам
- ежедневный progress update
- weekly summary
- reminders по каждой цели

- Telegram Bot API
- один backend
- PostgreSQL
- S3-compatible storage
- OpenAI API
- scheduler / background jobs
- GitHub для кода и документации
